## Microprojeto - Estatística Básica em Python

### Objetivo
- Implementar cálculos estatísticos simples (média, variância, desvio padrão)
- Aplicar em um dataset fictício (ex.: notas de alunos, temperaturas, vendas)
- Publicar no GitHub com documentação clara

---

### Estrutura do Repositório

estatistica-basica-python/
│
├── data/                # datasets fictícios (CSV ou JSON)
├── notebooks/           # Jupyter Notebooks com análises
├── src/                 # scripts Python organizados
│   ├── estatistica.py   # funções de média, variância, etc.
│   └── utils.py         # funções auxiliares
├── tests/               # testes unitários (pytest)
├── README.md            # documentação do projeto
├── LICENCE              # licença
└── requirements.txt     # dependências (numpy, pandas, matplotlib)

---

## 📑 Boas práticas adotadas

- Notebooks com seções claras e narrativas em *markdown*.  
- Funções auxiliares em scripts (`src/`).  
- Dados organizados em pastas (`raw` e `processed`).
- Dependências listadas em `requirements.txt`.  
- Licença aberta para uso e colaboração.
